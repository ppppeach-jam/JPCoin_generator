# JPCoin_generator

ランダムな4種の背景に小銭を配置した画像を生成します。
画像および画像内の各小銭の枚数、合計金額を記載した.txtファイルを出力します。

## ディレクトリ構成
```
assets
├── 1yen
      ├─ back.png
      ├─ front.png
├── 5yen
      ├─ ...
├── 10yen
├── 50yen
├── 100yen
├── 500yen
backgrounds
├── black.png
├─ ...
output
├──images
├──labels
main.py
```
